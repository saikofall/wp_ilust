シンプルでカスタマイズ可能なWordPressテーマ

== Description ==

「マイテーマ」は、ブログやウェブサイトに最適なシンプルで使いやすいWordPressテーマです。
レスポンシブデザインに対応し、モバイルデバイスでも快適に閲覧できます。

== 特徴 ==

* レスポンシブデザイン
* カスタムロゴのサポート
* カスタム背景のサポート
* ウィジェット対応（サイドバー、フッター3カラム）
* ナビゲーションメニュー対応
* アイキャッチ画像のサポート
* HTML5マークアップ
* 翻訳対応

== インストール ==

1. WordPressの管理画面から「外観」→「テーマ」→「新規追加」をクリック
2. 「テーマのアップロード」をクリック
3. zipファイルを選択してアップロード
4. 「有効化」をクリック

または、

1. テーマファイルを /wp-content/themes/ ディレクトリにアップロード
2. WordPressの管理画面から「外観」→「テーマ」で有効化

== カスタマイズ ==

「外観」→「カスタマイズ」から以下の設定が可能です：

* サイトのタイトルとキャッチフレーズ
* カスタムロゴ
* カスタム背景
* メニュー設定
* ウィジェット設定

== よくある質問 ==

= このテーマはレスポンシブ対応ですか？ =

はい、レスポンシブデザインに完全対応しています。

= ウィジェットエリアはいくつありますか？ =

サイドバーに1つ、フッターに3つのウィジェットエリアがあります。

= メニューはいくつ設定できますか？ =

プライマリーメニューとフッターメニューの2つを設定できます。
